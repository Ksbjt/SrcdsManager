# SrcdsManager
### Usage 用法

- 打开 xnet_srcdsmanager.exe

### Language 语言偏好

`Setting --> Language --> Set Language File --> Save`

- chi.ini
- English.ini
- French.ini
- Korean.ini
- Polish.ini
- Spain.ini
- zho.ini

### Function 功能

- 计划任务（定制重启，更新……等等）
- 自动更新服务器（不过要从软件中下载服务端，否则请将自动更新取消勾选）
- 崩溃自动重启
